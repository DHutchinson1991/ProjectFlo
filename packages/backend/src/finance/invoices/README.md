# Invoices Module

## What this module does

CRUD for invoices scoped to an inquiry. Supports manual invoice creation with line items and auto-generation of deposit invoices when a contract is signed. Total amount is computed server-side from items.

## Key files

| File | Purpose |
|------|---------|
| `invoices.service.ts` | CRUD + `autoGenerateDepositInvoice` |
| `invoices.controller.ts` | `POST/GET/PATCH/DELETE /api/inquiries/:inquiryId/invoices` |
| `dto/create-invoice.dto.ts` | Validation: items array, dates, status |
| `dto/update-invoice.dto.ts` | Partial update DTO |

## Business rules / invariants

- `amount` is computed server-side from `items.reduce(qty × unit_price)` — frontend never sends total.
- `autoGenerateDepositInvoice` resolves deposit from: primary quote `deposit_required` → quote `total_amount` → selected package `base_price` → 0.
- Auto-generated deposit invoice has status `Sent` and 7-day due date.
- Auto-generation also completes the "Raise Deposit Invoice" inquiry task.
